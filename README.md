# Romanticwave

Romanticwave é uma interface estática para criar cartas românticas personalizadas com visual premium, pensada para GitHub Pages ou qualquer hospedagem estática.

## O que já funciona

- Tela inicial elegante e responsiva
- Botão `+` abre o menu de cartas
- Editor com frente e verso
- Personalização em tempo real
- Papel, textura, tamanho, fonte, cor e alinhamento
- Adesivos e fotos com arrastar e soltar
- Rascunho salvo no navegador
- Envio de teste por ID
- Caixa de recebimento por ID no próprio navegador
- Link compartilhável do conteúdo da carta

## Como usar

1. Abra `index.html` no navegador.
2. Clique em `+` para abrir o menu.
3. Crie ou retome uma carta.
4. Personalize e clique em **Salvar e enviar por ID**.
5. Use a caixa de teste para abrir cartas por ID.

## Publicação no GitHub

Basta subir estes arquivos:

- `index.html`
- `style.css`
- `script.js`

Se estiver usando GitHub Pages, escolha a branch principal e a raiz do repositório.

## Observação importante

O envio por ID aqui é uma simulação local com armazenamento do navegador.  
Para envio real entre dispositivos e contas diferentes, será necessário adicionar um backend depois.
